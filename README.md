# E-Commerce Intelligence Data Analysis (Jarvis Hackathon)

This project analyzes enterprise e-commerce datasets using PySpark in Databricks.

## Goal
Identify patterns in customer distribution across countries to inform marketing strategy.

## Tools
- PySpark
- Databricks
- Data Visualization

## Dataset
Customer and transaction datasets were analyzed to compute metrics such as:
- total revenue
- number of customers
- average loyalty score
- average user age
- average lifetime value

These metrics were grouped by country to uncover regional trends.
